## Core insight

**Replay captures what actually happened.**

Not logs. Not stack traces. A **deterministic recording** of every DOM change, network request, and state update — the full runtime picture the site says agents have not had access to before.

### Four-step story (as shown on the page)

1. **Browser session** — A bug occurs in the browser  
2. **Replay recording** — Every detail is captured  
3. **Agent gets context** — Root cause is identified  
4. **Fix shipped** — Your agent applies the fix
